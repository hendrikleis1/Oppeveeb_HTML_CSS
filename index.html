Sissejuhatus

Mis on HTML?

HTML ehk HyperText Markup Language on veebilehtede loomise aluskeel. See ei ole programmeerimiskeel, vaid märgistuskeel, mis kirjeldab lehe sisu ja struktuuri: millised osad on pealkirjad, millised lõigud, kus on pildid, lingid, tabelid või vormid. HTML kasutab silte (tage), näiteks <h1> suurima pealkirja või <p> lõigu jaoks, et anda sisule tähendus ja roll. Korralik HTML-struktuur aitab brauseritel, otsingumootoritel ja abitehnoloogiatel sisu õigesti tõlgendada, mis parandab nii loetavust kui ka ligipääsetavust. Kujundus ja värvid lisatakse tavaliselt eraldi stiililehel (CSS), et hoida sisu ja esitus lahus. Kokkuvõttes on HTML veebilehe vundament: see määrab, mis lehel on ja kuidas see on üles ehitatud.

Mis on CSS?

CSS (Cascading Style Sheets) on stiililehtede keel, millega kujundatakse HTML-is loodud veebilehti. Kui HTML määrab, milline sisu lehel on, siis CSS ütleb, kuidas see sisu välja näeb – näiteks värvid, fondid, suurused, paigutus ja animatsioonid. CSS võimaldab eraldada kujunduse ja struktuuri: sama HTML-sisu saab kuvada väga erineva välimusega lihtsalt stiilireegleid muutes. Reeglid koosnevad valijatest (nt elemendi nimi, klass või ID) ja omadustest (nt color, margin, display). Tänu CSS-ile on võimalik luua paindlikke, reageerivaid (RWD) ja ligipääsetavaid kujundusi, mis kohanduvad erinevate ekraanide ja seadmetega.

Teooria

HTML teooria

Iga element koosneb tavaliselt avavast sildist, sisust ja sulgevast sildist. Näiteks: 

<p>See on üks lõik teksti.</p>

HTML-i põhielemendid jaotatakse tavaliselt järgmistesse kategooriatesse:

  • Pealkirjad – <h1> kuni <h6>
  • Lõigud ja tekst – <p>, <span>
  • Lingid – <a href=\"...\">
  • Pildid – <img src=\"...\" alt=\"...\">
  • Loendid – <ul>, <ol>, <li>
  • Tabelid – <table>, <tr>, <td>
  • Vormid – <form>, <input>, <button>

Lisaks on oluline lisada metaandmed (nt märgistus <meta charset=\"utf-8\"> või lehe pealkiri <title>) dokumendi <head>-i osasse.

Kokkuvõttes on HTML veebilehe vundament: see määrab, mis elemendid lehel on ja kuidas need on loogiliselt üles ehitatud. Kujundus ja värvid lisatakse CSS-iga, dünaamiline käitumine aga JavaScriptiga.

CSS teooria – põhitõed

CSS (Cascading Style Sheets) on keel, mis määrab HTML-elemendi välimuse – värvid, fondid, vahed, piirjooned ja paigutuse. CSS eraldab kujunduse sisust, muutes koodi lihtsamini hallatavaks.

Stiilireegel koosneb valijast ja omadustest. Näiteks: 

p { color: #222; margin-bottom: 1rem; }
  • Elemendivalija: p
  • Klass: .btn
  • ID: #pealkiri
  • Pseudoklass: :hover

Paigutuseks kasutatakse Flexboxi (joondus reas või veerus) ja Gridi (keerukamad skeemid). Mobile-first lähenemisel kirjutatakse stiilid esmalt väikestele ekraanidele ja seejärel lisatakse @media abil suuremate ekraanide täiustused.
